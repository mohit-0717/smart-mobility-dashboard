# Smart Mobility Traffic Dashboard 🚦

## 📊 Overview
This project is an IoT-based dashboard developed using HTML, CSS, JavaScript, and Chart.js. It visualizes smart city traffic data including congestion, EV charging, accidents, and public transport.

## 🔗 Dataset
Smart Mobility Traffic Dataset (Kaggle)  
https://www.kaggle.com/datasets/ziya07/smart-mobility-traffic-dataset

## 🤖 AI Tool Used
Claude AI (Anthropic)

## 📌 Features
- Traffic congestion prediction & analysis
- EV charging station analytics
- Road accident prediction & visualization
- Public transport efficiency tracking
- Interactive filters (zone, weather, hour)
- Live congestion heatmap
- Dataset preview table (sorting & pagination)
- Dark/Light mode toggle
- Export dashboard to PDF

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Chart.js
- AI-assisted development
